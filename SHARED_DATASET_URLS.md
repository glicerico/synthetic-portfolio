# Shared Dataset URLs

The following URLs point to the public dataset packages (zip files) generated by the synthetic portfolio benchmark. These links are intended for use by agents and evaluators. All URLs are hosted on catbox.moe.

| Variant | Difficulty | Download URL |
|---|---|---|
| Easy | Easy | https://files.catbox.moe/as543n.zip |
| Medium | Medium | https://files.catbox.moe/143ds6.zip |
| Hard | Hard (Regime Shift) | https://files.catbox.moe/03klu4.zip |
| Noise Trap | Trap (No alpha over EW) | https://files.catbox.moe/sdttci.zip |
| Turnover Trap | Trap (High costs erase return) | https://files.catbox.moe/abtv2t.zip |

*These URLs are refreshed each time `python -m portfolio_eval generate_all --share` is run.*
